## Creating Your First Creative

### Step 1: Start a New Project
1. Click "New Project" from the dashboard
2. Enter a project name (e.g., "Summer Sale Campaign")
3. Select a format:
   - **Social Media:** Facebook, Instagram, LinkedIn
   - **Display Ads:** Banners, rectangles
   - **Custom:** Define your own dimensions

### Step 2: Upload Assets
1. Click "Assets" in the sidebar
2. Drag and drop images or click to browse
3. Supported formats: JPG, PNG, WebP
4. Max file size: 10MB per image

### Step 3: Design Your Creative
1. **Add Elements:**
   - Click toolbar icons to add text, shapes, or images
   - Drag elements onto the canvas
   - Resize and position as needed

2. **Edit Properties:**
   - Select an element
   - Use the properties panel to adjust:
     - Colors
     - Fonts
     - Sizes
     - Opacity

3. **Layer Management:**
   - View all elements in the layers panel
   - Drag to reorder
   - Click eye icon to show/hide
   - Lock elements to prevent editing

## Compliance Validation

### Running a Compliance Check
1. Click "Compliance" in the top bar
2. Click "Check" to validate your creative
3. Review the compliance report:
   - **Score:** Overall compliance percentage
   - **Errors:** Critical issues that must be fixed
   - **Warnings:** Recommendations for improvement

### Understanding Violations
Each violation shows:
- **Rule Name:** What guideline was violated
- **Message:** Clear explanation of the issue
- **Severity:** Error (must fix) or Warning (should fix)

### Using the AI Copilot
1. Click "Ask Copilot" on any violation
2. Get AI-powered guidance:
   - **Explanation:** Why this rule exists
   - **Business Context:** Impact on campaign
   - **Fix Suggestion:** How to resolve it
   - **Learning Tip:** Avoid future issues

### Applying Fixes
1. Review the suggested fix
2. Click "Apply Fix" (if available)
3. Or manually adjust your creative
4. Re-run compliance check to verify

### Certification
1. Ensure all errors are resolved
2. Click "Certify" button
3. Receive a compliance certificate
4. Download certificate for records

## Tips & Best Practices

### Design Tips
1. **Start with Templates:** Use AI-generated layouts as a starting point
2. **Maintain Hierarchy:** Make important elements larger and bolder
3. **Use White Space:** Don't overcrowd your design
4. **Brand Consistency:** Stick to your brand colors and fonts

### Compliance Tips
1. **Check Early:** Run compliance checks throughout design process
2. **Safe Zones:** Keep critical elements away from edges
3. **Logo Size:** Ensure logos meet minimum size requirements
4. **Text Readability:** Use high contrast for text on backgrounds

### Workflow Tips
1. **Auto-Save:** Your work is automatically saved every 30 seconds
2. **Keyboard Shortcuts:**
   - `Ctrl/Cmd + Z`: Undo
   - `Ctrl/Cmd + Y`: Redo
   - `Delete`: Remove selected element
3. **Preview Often:** Use the Preview button to see final result
4. **Organize Assets:** Use descriptive names for easy finding

### Performance Tips
1. **Optimize Images:** Compress images before uploading
2. **Limit Elements:** Too many elements can slow down the editor
3. **Use Layers:** Organize complex designs with layers
4. **Save Regularly:** Although auto-save is enabled, manual saves are instant

## Troubleshooting

### Issue: Canvas Not Loading
**Solution:**
- Refresh the page
- Clear browser cache
- Try a different browser (Chrome recommended)

### Issue: AI Generation Failed
**Solution:**
- Check your internet connection
- Simplify your prompt
- Try again in a few moments

### Issue: Export Taking Too Long
**Solution:**
- Reduce number of selected formats
- Check file sizes of uploaded images
- Ensure stable internet connection

### Issue: Compliance Check Errors
**Solution:**
- Read error messages carefully
- Use AI Copilot for guidance
- Refer to brand guidelines
- Contact support if unclear
