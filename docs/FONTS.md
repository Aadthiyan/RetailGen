## ‚ö° **Pro Tips:**

1. **Contrast is Key:** Pair bold display fonts with clean sans-serif
2. **Size Hierarchy:** Headline 3-5x larger than body text
3. **Limit Fonts:** Use max 2-3 fonts per poster
4. **Readability:** Display fonts for headlines only, never body text
5. **Test Distance:** Posters should be readable from 10+ feet away
